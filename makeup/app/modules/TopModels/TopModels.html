Below you can experiment with the CRUD functions of the SampleData module. It uses the Service SampleService, which you find in the "services" folder. A Service is used to retrieve and manipulate data of a Database. Note how the SampleData module is nested in the TopModels module with the following line in the template file:

<div data-mod="SampleData">[[SAMPLE_DATA]]</div>
